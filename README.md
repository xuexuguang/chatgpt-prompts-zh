## 桌面版使用  

1. 下载 [chatgpt桌面版](https://github.com/lencx/ChatGPT/blob/main/README-ZH_CN.md)
2. 点击菜单栏的contrl center
3. 点击language model 
2. 点击sync custom 
3. 添加源：raw.githubusercontent.com/xuexuguang/chatgpt-prompts-zh/master/chatgpt-prompts-zh
